Files & Directories

HTML File Included

  • index-image.html is for image style Home Page
  • index-particles.html is for particles style Home Page
  • index-video.html is for video style Home Page
  • index-type.html is for Typing style Home Page
  • index-slideshow.html is for Slideshow style Home Page
  • index-round.html is for special style Home Page
  • single-post.html is for single blog post

Template Installation

1. FTP Upload:

  1. Open up your FTP manager and connect to your hosting
  2. Browse to required directory (Normally public_html)
  3. Upload the files inside HTML folder.

HTML Structure

We are using Bootstrap framework. So, basic HTML structure is same as it is used in Bootstrap.

CSS & Javascripts

CSS Files Structure

    <!-- ========== Bootstrap & Other CSS Files ========== -->

    <link rel="stylesheet" href="css/font-awesome.min.css">
    <link rel="stylesheet" href="css/magnific-popup.css">
    <link rel="stylesheet" href="css/owl.carousel.css">
    <link rel="stylesheet" href="css/animate.min.css">

    <!-- ========== Main Style CSS ========== -->
    <link rel="stylesheet" href="style.css">

Javascript Files Structure

<!-- ========== Include jQuery ========== -->
    <script src="jquery.min.js"></script>

    <!-- ========== Other JS ========== -->
    <script src="js/jquery.parallax.js"></script>
    <script src="js/plugin.js"></script>
    <script src="js/init-particles.js"></script><!-- Required for particles background -->


    <!-- ========== Main js file ========== -->
    <script src="js/main.js"></script>
